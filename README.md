# Currency CLI (API_2)

Мини-проект для получения курсов валют через `open.er-api.com`, кэширования в файл и конвертации суммы между валютами.

## Структура проекта

- `currency.py` — точка входа (запускает CLI).
- `cli.py` — интерфейс пользователя и сценарии работы.
- `api_client.py` — HTTP-запросы к API.
- `storage.py` — чтение/запись JSON и проверка свежести кэша.
- `currency_rate.json` — кэш курсов.
- `test_currency.py` — базовые автотесты.
- `legacy/` — файлы предыдущего учебного проекта (`main.py`, `http_client.py`).

## Что умеет программа

- Запрашивает базовую валюту (например, `USD`).
- Показывает курсы для `RUB`, `EUR`, `GBP`.
- Проверяет кэш в `currency_rate.json`:
  - если файлу меньше 24 часов — читает из файла;
  - иначе — обновляет данные через API.
- Конвертирует сумму: `из_валюты -> в_валюту`.
- Валидирует коды валют по списку ключей `rates.keys()`.
- Выводит результат конвертации с 4 знаками после запятой.

## Запуск

```powershell
.\venv\Scripts\activate
python currency.py
```

## Пример сценария

1. Ввести базовую валюту: `USD`
2. Ввести для конвертации:
   - из валюты: `USD`
   - в валюту: `EUR`
   - сумма: `100`
3. Получить результат:
   - `100.0000 USD = ... EUR`
